Spring Core Annotations

@Autowired

以下三种写法，其效果是一样，实际使用哪个看个人喜好

  • Constructor injection
  • Setter injection:
  • Field injection:
@Bean

所有使用@Bean注释的方法都必须在@Configuration类中。

@Qualifier

需要和@Autowired搭配使用。

@Required

适用于bean属性setter方法，并表示受影响的bean属性必须在XML配置文件在配置时进行填充。

@Value
  • Constructor injection:
  • Setter injection:
  • Alternatively:
  • Field injection:
@DependsOn

控制bean加载顺序,当eventListener初始化后，eventListener才能初始化，当然也可以在eventListenerBean方法上增加@Lazy可以达到同样效果。
